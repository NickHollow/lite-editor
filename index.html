Lite Editor

Lite Editor is a simple JavaScript WYSIWYG Editor
focused on inline elements such as strong em a i

Quick start

Using CDN

You can easily start by using CDN.

Add the following description in the <head> tag of the web page you want to use it.

<script src="https://unpkg.com/lite-editor@1.0.5/js/lite-editor.min.js"></script>
<link rel="stylesheet" href="https://unpkg.com/lite-editor@1.0.5/css/lite-editor.css">
new LiteEditor('.js-lite-editor');
