Quick start

Using CDN

You can easily start by using CDN.

Add the following description in the <head> tag of the web page you want to use it.

<script src="https://unpkg.com/simple-wysiwyg@0.5.1/js/simple-wysiwyg.min.js"></script>
<link rel="stylesheet" href="https://unpkg.com/simple-wysiwyg@0.5.1/css/simple-wysiwyg.css">
new SimpleWysiwyg('.js-simple-wysiwyg');
