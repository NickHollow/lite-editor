Methods

Methods are actions done by LiteEditor instances. Vanilla JS methods look like editor.methodName( /* arguments */ ). Below is the example of the code.

editor.redo();

registerButton (btnOption)

Add a new button to the editor.

saveSelection ()

Save a selected area.

restoreSelection ()

Restore a selected area.

insertTag (tag, className, sampleText)

Insert a tag to a selected area.

undo ()

Go back to the previous state.

redo ()

Go forward to the next state.
